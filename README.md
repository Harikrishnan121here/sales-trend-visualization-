# Sales Trend Visualization Project

## Objective
The objective of this project is to analyze motor vehicle sales trends using data analytics and visualization techniques.

## Dataset Used
Colorado Motor Vehicle Sales Dataset

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
- GitHub

## Tasks Performed
- Data Cleaning
- Data Analysis
- Sales Trend Visualization
- County-wise Sales Analysis
- Quarter-wise Sales Distribution

## Visualizations Included
1. Yearly Sales Trend Line Chart
2. Top County Sales Bar Chart
3. Quarter-wise Sales Pie Chart

## Key Insights
- Vehicle sales fluctuated across different years.
- Some counties generated significantly higher sales.
- Sales distribution varied across quarters.

## Conclusion
This project demonstrates how data analytics and visualization techniques can be used to identify sales patterns and business insights.
